In today's data-driven economy, the ability to effectively import, clean, and manage vast quantities of data is not just a technical skill but a fundamental requirement for business success. As organizations grapple with the deluge of information generated by IoT devices, social media, and transactional systems, the infrastructure for data ingestion has become increasingly complex and critical. This necessity is what drives the intense demand for proficiency in core analytical tools: SQL remains indispensable for querying and structuring relational databases; Python serves as the versatile engine for advanced statistical modeling, machine learning, and automated data pipelines; and data analysis itself is the overarching skillset that transforms raw data into actionable business intelligence. Consequently, professionals skilled in combining these three areas—data importing, SQL querying, Python programming, and rigorous data analysis—are highly sought after across virtually every sector, making these competencies essential currency in the modern job market.

Where did those numbers come from?

It was a long process but here are the steps that were taken:

  • ◌ Extract job listings using AWS ECS on a schedule
  • ◌ Store them in a Postgres DB on AWS
  • ◌ Use Gemma4 to extract the job skills for each job description
  • ◌ Use FAISS (Facebook AI Similarity Search) to create clusters of extracted job skills
  • ◌ Use Gemma4 to label the clusters created by FAISS
  • ◌ Map the labels back to the extracted job skills
  • ◌ Calculate the frequencies of each cluster across all job postings
